## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites

- NodeJS - [v16.x](https://nodejs.org/en/)
- [Yarn](https://yarnpkg.com/)

### Setup

```bash
git clone https://github.com/omodauda/ContactBook.git
cd ./ContactBook
```

### Install

```bash
yarn install
```

### Usage

```bash
yarn start
```

### Build

```bash
yarn android
yarn ios
```
